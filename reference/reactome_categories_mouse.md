# Mouse pathway hierarchy from Reactome

Mouse pathway hierarchy from Reactome

## Usage

``` r
reactome_categories_mouse
```

## Format

A data frame (tibble) with 1748 rows and 4 columns:

- id:

  Pathway ID

- description:

  Pathway name

- level_1:

  Top-level parent term

- level_2:

  Second-level hierarchy term

## Source

<https://reactome.org/>

## Value

An object of class "tbl_df", "tbl", "data.frame"
