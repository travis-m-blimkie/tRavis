# Human gene ID mapping from biomaRt

Human gene ID mapping from biomaRt

## Usage

``` r
biomart_id_mapping_human
```

## Format

A data frame (tibble) with 76064 rows and 3 columns:

- ensembl_gene_id:

  Human Ensembl gene IDs

- hgnc_symbol:

  Human gene symbols

- entrez_gene_id:

  Human NCBI/Entrez gene IDs

## Source

<https://bioconductor.org/packages/biomaRt/>

## Value

An object of class "tbl_df", "tbl", "data.frame"
