# Mouse gene ID mapping from biomaRt

Mouse gene ID mapping from biomaRt

## Usage

``` r
biomart_id_mapping_mouse
```

## Format

A data frame (tibble) with 57548 rows and 3 columns:

- ensembl_gene_id:

  Mouse Ensembl gene IDs

- mgi_symbol:

  Mouse gene name/symbol

- entrez_gene_id:

  Mouse NCBI/Entrez gene IDs

## Source

<https://bioconductor.org/packages/biomaRt/>

## Value

An object of class "tbl_df", "tbl", "data.frame"
